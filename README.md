# 純前端背單字應用

這是一個使用 HTML、CSS、JavaScript 實作的純前端背單字應用。沒有使用任何前端框架，資料儲存在瀏覽器 `localStorage`。

## 功能

- 學習模式：顯示單字卡片，點擊卡片可翻面查看翻譯、詞性、例句、字根分析。
- 管理模式：新增、編輯、刪除單字資料。
- 自動填入：輸入英文單字後，按「自動填入」會呼叫公開 API 自動填入翻譯、詞性及例句。

## 使用方式

1. 開啟 `index.html`。
2. 在上方切換「學習」與「管理單字」。
3. 在管理頁面新增單字資料，或使用自動填入按鈕快速補資料。

## 注意

- 若單字 API 查無資料，仍可手動補齊各欄位。
- 單字資料會儲存在當前瀏覽器，刷新頁面後資料仍會保留。
